# 🐺 AI 狼人杀 (Serverless / LangChain Driven)

## 🎮 简介

这是一个由 **LangChain 驱动的 AI 狼人杀游戏**。
你可以直接在浏览器中体验一场完整的狼人杀对局：

- 玩家扮演其中一个角色；
- 其余角色由语言模型（AI）扮演；
- 游戏包含 **昼夜循环、发言、投票、阵营对抗、胜负判定** 等完整逻辑；
- 支持角色：狼人、预言家、女巫、猎人、村民。

无需后端支持，所有游戏逻辑与 AI 推理均在本地完成。

---

## 🧠 核心实现

本项目使用 **LangChain + LangGraph** 实现 AI 回合编排逻辑：

- 每个角色是一个独立的智能体（Agent）；
- LangGraph 负责管理阶段流转（夜晚行动 / 白天发言 / 投票 / 判定）；
- 所有智能体的对话、记忆与推理均通过 LangChain 的标准链式调用完成。

这意味着你能看到 **模型在复杂交互场景中如何推理、合作与冲突**。

---

## 🔐 无服务端设计（Serverless）

- 项目完全在 **浏览器端运行**，不依赖任何后端代理或服务器。
- 你只需 **填入自己的 OpenAI API Key** 即可开始游戏。
- API Key 不会上传、不会存储、不会经过任何中转服务器。
- 所有请求均直接从你的浏览器发送到 OpenAI API，**零泄露风险**。

---

## 🧪 模型评测能力

除了是一款轻量有趣的文字游戏，这个项目还可作为
**语言模型在复杂对话与推理场景下的评测工具**：

- 测试模型的 **多角色推理能力**（同时维持多重身份与目标）。
- 观察模型在 **高信息噪声和社交博弈场景** 下的表现。
- 分析模型的 **幻觉（hallucination）现象** 和一致性问题。
- 用于比对不同模型（如 GPT、Claude、Gemini）在复杂语境下的策略差异。

## 模型评测记录

### deepseek-chat

1、幻觉严重，通常在第一晚行动中会出现总结上一轮的发言情况而做出选择
2、信息不明确，在prompt中明确说明目前仅存活的玩家后，在后续的操作中仍然会选择已死亡玩家进行操作

## 🚀 快速开始

1. 打开游戏页面
2. 在设置界面中输入你的 **OpenAI API Key**
3. 选择角色并点击「开始游戏」
4. 享受一场由 AI 主导的狼人杀对局 🎭

## 📚 技术栈

- **Vue 3**：UI 与状态展示
- **LangChain / LangGraph**：AI流程编排与推理逻辑
- **OpenAI API**：自然语言决策与对话输出
- **Serverless 前端架构**：浏览器本地执行，无任何后端依赖
